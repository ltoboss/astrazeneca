# Challenges de Entrevista de Trabajo

## Problema 1: Instrucciones Encriptadas (JavaScript)

Este challenge fue implementado utilizando JavaScript y se puede ejecutar simplemente abriendo el archivo HTML proporcionado. Además, requiere un archivo de texto (ejemplo: "input.txt") que contiene las instrucciones a ser procesadas.

### Instrucciones para ejecutar el Problema 1:

1. Navega a la carpeta "Instrucciones Encriptadas".
2. Selecciona un archivo de texto que contenga la información con la que el programa trabajará (en dado caso, existe una ya en la carpeta)
3. Abre el archivo "index.html" en tu navegador web.
4. El resultado del programa se mostrará en la consola del navegador y se guardará en un archivo de texto llamado "results.txt".

## Problema 2: Jugador Ganador (JavaScript)

En este challenge, se ha implementado el mismo problema en dos lenguajes de programación diferentes: JavaScript y Python. Ambos requieren un archivo de texto (ejemplo: "input.txt") para obtener los datos de entrada y generan un archivo de texto llamado "result.txt" con los resultados.

### Instrucciones para ejecutar el Problema 2 (JavaScript):

1. Navega a la carpeta "Jugador Ganador".
2. Selecciona un archivo de texto que contenga la información con la que el programa trabajará (en dado caso, existe una ya en la carpeta)
3. Abre el archivo "index.html" en tu navegador web.
4. El resultado del programa se mostrará en la consola del navegador y se guardará en un archivo de texto llamado "result.txt".

### Instrucciones para ejecutar el Problema 2 (Python):

1. Navega a la carpeta "Jugador Ganador Python".
2. Coloca el archivo de texto "input.txt" con los datos de entrada en la misma carpeta.
3. Asegúrate de tener Python instalado en tu sistema.
4. Ejecuta el archivo "jugador_ganador.py" desde tu terminal o símbolo del sistema con el comando:
   ```bash
   python jugador_ganador.py
   ```
5. El resultado del programa se mostrará en la consola y se guardará en un archivo de texto llamado "result.txt".
